### IF :

    - syntx: if(condition){
    	statement;
    	}

### IF-ELSE

    - syntax:
    if(condition)
    {
    	statement;
    }
    else
    {
    	statement;
    }

### IF-ELSEIF-ELSE

    -syntax:
    if(condition)
    {
    	statement;
    }
    else if(condition2)
    {
    statement;
    }
    else
    {
    	statement;
    }

### Loop

    -syntax:
    loop
    {
    	statement;
    	if(condition)  // Otherwise the loop will be infinite
    	{
    		break;
    	}
    }

### For Loop

    - syntax:

    for var in expression
    {
    	statement;
    }

    // expression for array with name a `a.iter()`

### While loop

    - syntax:
    	while(condition)
    	{
    		statement;
    	}
