### Structure:

    - User defined data type that can consisit variable of different types
    - Syntax:
    	struct name
    	{
    		variable1: data-type,
    		variable2: data-type,
    		variable3: data-type
    	}

### Enum:

    - User defined data types with definite values
    - Syntax:
    	enum name
    	{
    		variable1,
    		variable2,
    		variable3,
    	}
